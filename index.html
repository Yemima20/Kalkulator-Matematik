Segitiga Jajar Genjang

Keliling Segitiga

Rumus keliling segitiga, yaitu:

K = S1 + S2 + S3

Di mana:

K= Keliling

S1 = Sisi A

S2 = Sisi B

S3 = Sisi C

Hitung Keliling Segitiga

K = S1 + S2 + S3

Luas Segitiga

Rumus luas segitiga, yaitu:

L = 1/2 x a x t

Di mana:

L= Luas

a = Panjang alas

t = Tinggi

Hitung Luas Segitiga

L = 1/2 x a x t

Keliling Jajar genjang

Rumus keliling jajar genjang, yaitu:

K = 2(a + b)

Di mana:

K= Keliling

a = Panjang alas

b = Sisi miring

Hitung Keliling Jajar Genjang

K = 2(a + b)

Luas Jajar genjang

Rumus luas jajar genjang, yaitu:

L = a x t

Di mana:

L= Luas

a = Panjang alas

t = Tinggi

Hitung Luas Jajar Genjang

L = a x t
